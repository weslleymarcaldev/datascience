Ferramentas

Ferramentas de Iniciante

Ferramentas essenciais para começar sua jornada em Data Science:

Python Pandas NumPy Matplotlib Seaborn Jupyter Notebook SQL PostgreSQL MySQL Git GitHub VS Code
Por que essas ferramentas?
  • Python: Linguagem mais usada em Data Science
  • Pandas: Manipulação de dados tabulares
  • NumPy: Operações matemáticas com arrays
  • Matplotlib/Seaborn: Visualização de dados
  • Jupyter: Ambiente interativo para análise
  • SQL: Consultar e manipular bancos de dados
  • Git: Versionamento e colaboração
Ferramentas Avançadas

Ferramentas para projetos mais complexos e produção:

Scikit-learn TensorFlow PyTorch FastAPI Docker Kubernetes Apache Spark Airflow AWS GCP Azure MLflow DVC Plotly Streamlit
Quando usar?
  • Scikit-learn: Machine Learning clássico
  • TensorFlow/PyTorch: Deep Learning
  • FastAPI: Deploy de modelos como API REST
  • Docker: Containerização e reprodutibilidade
  • Spark: Big Data e processamento distribuído
  • Airflow: Orquestração de pipelines ETL
  • Cloud: Infraestrutura escalável
  • MLflow: Tracking e gestão de experimentos
  • Streamlit: Apps de dados interativos
